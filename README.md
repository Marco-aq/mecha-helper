## 📖 About the Game
You are a modular repair drone inside a massive, chaotic machine. The pressure is rising, and critical failures are appearing everywhere! Equip the right modules to fix the specific failures before the Core reaches 100% pressure and explodes.

### Controls & Modules
* **Arrow Keys**: Move and Jump
* **[ 1 ] Base Module**: Normal movement.
* **[ 2 ] Jetpack Module**: Hold `[UP]` to fly. Used to extinguish Aerial Fires 🔥.
* **[ 3 ] Tread Module**: High ground speed. Used to repair Ground Gears ⚙️.

## 🚀 How to run locally
1. Clone this repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local server.
4. Open `localhost:5173` (or the port provided by Vite).
